QoE-aware Networking
====================

Overview
--------

Quality of Experience (QoE) aware networking shifts focus from network-centric metrics (throughput, latency) to user-perceived application quality.

.. note::
   This project develops QoE-driven resource allocation strategies for EdgeRIC.

The QoE Gap
-----------

Traditional network optimization targets:

- Maximize throughput
- Minimize latency
- Ensure fairness

But users care about:

- Video stall duration and frequency
- Application responsiveness
- Perceived quality consistency

Our Approach
------------

QoE Estimation
^^^^^^^^^^^^^^
Real-time inference of application-level QoE from network observables.

QoE-driven Scheduling
^^^^^^^^^^^^^^^^^^^^^
Resource allocation that directly optimizes user experience metrics.

Multi-application QoE
^^^^^^^^^^^^^^^^^^^^^
Balancing QoE across diverse concurrent applications.

Supported Applications
----------------------

* **Video Streaming**: Minimize rebuffering, maximize quality
* **Video Conferencing**: Optimize for interactivity and clarity
* **Web Browsing**: Reduce page load times
* **Gaming**: Minimize input lag and frame drops
