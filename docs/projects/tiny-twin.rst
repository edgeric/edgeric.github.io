Tiny-twin
==================

Overview
--------

Cloud gaming represents one of the most demanding applications for 5G networks, requiring ultra-low latency and high throughput to deliver seamless gaming experiences.

.. note::
   This project explores how EdgeRIC can optimize network resources for cloud gaming workloads in real-time.

Key Challenges
--------------

* **Latency Sensitivity**: Cloud gaming requires sub-20ms round-trip times for responsive gameplay
* **Variable Bitrate**: Game streams have highly variable bandwidth requirements
* **QoE Metrics**: Traditional network metrics don't capture gaming experience quality

Our Approach
------------

EdgeRIC enables real-time adaptation of scheduling and resource allocation based on:

- Frame timing and rendering deadlines
- Player input patterns
- Network conditions
